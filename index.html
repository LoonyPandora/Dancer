  • What is Dancer, and why should I use it?

    Dancer is a minimalistic perl web application framework. It provides an elegant, intuitive, and expressive syntax that allows you to get on with writing your app.

    It is a flexible Modern Perl framework with a modular design which includes support for PSGI and plugins â€“ allowing rapid development and trivial scaling

    It has few dependencies, only relying on a few non-core Perl modules. Meaning you can deploy it almost anywhere.
